Calgary Communities

CGyHood — Calgary Community Comparator

Calgary has over 200 communities — roughly the same geographic footprint as New York City — yet no simple tool existed to compare two of them side by side. CGyHood solves that: pick any two communities and the app overlays their boundaries on an interactive Google Map, then surfaces key demographics and statistics sourced directly from the City of Calgary's open data portal, making it easy to evaluate neighbourhoods for relocation or business planning.

Built as a full-stack collaboration with Alex Jinga, the project intentionally demonstrates the interaction of multiple technology tiers. The React front-end uses the Google Maps JavaScript API (via the react-google-maps library) and react-bootstrap to render community polygons as live map overlays and drive the comparison UI. Behind it, a Python / Flask REST API is the primary data layer, serialising records from a PostgreSQL database hosted on Amazon AWS RDS. A Node / Express / Knex server was added as a proof-of-concept alternate back-end, showing that the same front-end can consume JSON from either runtime. Community boundary polygons were originally distributed in ArcGIS format; a custom Python pipeline converted them to Google Maps-compatible coordinates before loading them into the database.

The stack reflects the full-spectrum skills developed through the EvolveU Full Stack Developer programme — JavaScript ES6, React, Python, Flask, Node.js / Express, PostgreSQL, and Amazon AWS — applied to a real civic problem rather than a contrived exercise.
